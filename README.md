# Проект: Прогноз Погоды

## Описание

Прогноз погоды. Поиск города и отображение нынешней погоды и прогноза на неделю. Данные берутся с https://open-meteo.com.

## Технологии

HTML / SCSS / БЭМ / Адаптивная верска / React / JavaScript / Webpack /
Работа с API

## Запуск проекта

npm run start
